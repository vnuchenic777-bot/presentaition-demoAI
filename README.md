# HTML-пакет для представления проекта

Открывать с главной страницы:

- `index.html` - управленческая база проекта с раскрывающимися разделами, HAR-доказательствами, планом запуска и промтом для следующего этапа.
- `demo.html` - отдельная демонстрационная страница будущего рабочего сайта: очередь работ, рекомендация ИИ, утверждение, перенос в БРС, ручная подпись и audit log.

Пакет не требует backend и может открываться как обычные локальные HTML-файлы.

В верхней панели и внутри левого меню есть кнопка с тремя линиями. Она скрывает и открывает боковое меню; меню теперь работает как overlay поверх страницы и не урезает ширину основного контента.

В `index.html` добавлен раздел `Глоссарий проекта`: поиск, фильтры по категориям и раскрывающиеся карточки с пояснениями терминов, англоязычных слов и ключевых тезисов.

В `demo.html` добавлены:

- автоматическое создание незаблокированной колонки БРС перед записью балла;
- ускоренный режим проверки со списком вымышленных работ, миниатюрами, компетенциями, баллами и кнопками расшифровки;
- массовая кнопка `Создать колонку, отправить и подписать пакет`;
- блок технической ИИ-интеграции: получение события, извлечение текста, Rule Pack, teacher approval, BRS resolve, batch write, verification, audit.
