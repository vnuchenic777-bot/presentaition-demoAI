# ФЭМ · MVP Bridge

Демонстрационный HTML-пакет проекта по внедрению ИИ-помощника для проверки студенческих работ и контролируемого переноса утвержденных результатов в БРС.

Проект показывает не просто идею “проверки работ с помощью ИИ”, а целевой управляемый контур: ИИ формирует рекомендацию, преподаватель утверждает академическое решение, а интеграционный bridge готовит перенос числового балла в незаблокированную ведомость БРС. Финальная ответственность и подпись остаются за преподавателем.

## Что можно посмотреть

- Управленческое описание проекта, его цели, ограничения и план запуска.
- HAR-аналитику по текущим сценариям работы `gtifem.ru` и `xfem.ru`.

- Центр документов для скачивания проектных материалов: HAR Gap Analysis, As-Is, To-Be, инженерная спецификация и рабочий план запуска.

## Зачем нужен этот MVP

Сейчас проверка работ, фиксация рецензии и перенос баллов между системами требуют ручных действий и создают риск ошибок. MVP предлагает аккуратный переход к автоматизации без потери преподавательского контроля.

Ключевая идея: автоматизировать подготовку, сопоставление и перенос данных, но не передавать ИИ право финального академического решения.

## Состав репозитория

- `index.html` - основная страница для представления проекта руководству.
- `demo.html` - интерактивная визуализация будущего рабочего сценария.
- `styles.css` - дизайн презентационного сайта и демо-кабинета.
- `app.js` - интерактивность: меню, фильтры, демо-сценарии, статусы и журнал.
- `docs/` - отдельные проектные документы в форматах DOCX, TXT, HTML и MD.
- `downloads/` - архив с полным пакетом материалов для скачивания одной кнопкой.
